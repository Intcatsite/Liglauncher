## 🔥 Особенности
* 🌑 **Dark Mode:** Интерфейс на CustomTkinter — глаза не вытекают ночью.
* ⚡ **Speed:** Запуск быстрее, чем Тоха допивает ликофей.
* 🛠 **Multi-Arch:** Работает на Ubuntu (.deb) и Windows (.exe).
